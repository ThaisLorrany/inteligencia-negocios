# 📊 Inteligência de Negócios (Business Intelligence)

## 📖 Sobre o Projeto

Este projeto consiste em um site informativo sobre **Inteligência de Negócios (Business Intelligence - BI)**, desenvolvido com HTML e CSS. O objetivo é apresentar conceitos fundamentais da área, demonstrando como os dados podem ser transformados em informações estratégicas para auxiliar a tomada de decisões nas organizações.

O site aborda diferentes aspectos do universo do BI, incluindo sua definição, as principais tecnologias utilizadas no mercado e as oportunidades de carreira para profissionais da área. Além disso, conta com animações e elementos visuais que tornam a navegação mais dinâmica e agradável.

## 🚀 Tecnologias Utilizadas

* HTML5
* CSS3
* Animações com CSS

## ✨ Funcionalidades

* Apresentação dos conceitos de Business Intelligence;
* Informações sobre tecnologias utilizadas na área de BI;
* Conteúdo sobre carreira e oportunidades profissionais;
* Navegação entre páginas temáticas;
* Interface moderna e intuitiva;
* Layout responsivo;
* Efeitos visuais e animações.

## 📄 Páginas do Site

### 🏠 Home (index.html)

Página inicial com uma visão geral sobre Inteligência de Negócios e acesso às demais seções do site.

### 📊 BI (bi.html)

Apresenta os principais conceitos de Business Intelligence, sua importância e aplicação no ambiente corporativo.

### 💻 Tecnologia (tecnologia.html)

Explora as tecnologias, ferramentas e recursos utilizados para análise de dados e geração de informações estratégicas.

### 🚀 Carreira (carreira.html)

Aborda as oportunidades profissionais, competências necessárias e perspectivas de carreira na área de BI.

## 🎯 Objetivo

Promover o conhecimento sobre Inteligência de Negócios, demonstrando sua relevância para empresas e profissionais que desejam utilizar dados de forma estratégica para alcançar melhores resultados.
